A comprehensive security suite for WordPress: WAF, Smart 404, Brute Force guard, custom Login URL, Anti-spam, File Guard (uploads), Integrity & Malware scan, Activity Log.

== Features ==
* Custom login URL (default /manager/), hides wp-login.php
* WAF with pattern detection and rate limiting
* Smart 404 guard (lockout abusive IPs)
* Brute Force protection (lockouts, whitelist)
* Anti-spam (honeypot, min submit time, link limits)
* File Guard (deny PHP in uploads, daily quick scan)
* Integrity check (core checksums), Malware quick scan
* Security headers & hardening
* REST & XML-RPC protection
* Activity logging (DB table + JSON export)
